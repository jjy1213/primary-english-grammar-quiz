# 题库文件夹

这个文件夹用于集中存放项目的原始题库资料、答案资料和后续整理结果。

## 当前内容

### 1. raw-exams

存放你提供的 2008 到 2021 年上海市中考英语原始 Word 试卷文件。

这些文件的文件名已经包含“试卷/试题及答案”或“试卷和答案”，因此当前先按“原始题目 + 原始答案同文件保存”处理。

### 2. index.csv

题库索引清单，用于快速查看每份资料的年份、格式、原始文件名，以及当前是否包含答案。

## 后续建议

后面如果你要我继续整理成可导入系统的结构，建议在这个文件夹下继续增加：

- `parsed/`：抽取后的纯文本或结构化中间结果
- `json/`：最终可导入项目的题库 JSON
- `answers/`：如果后续需要把答案单独拆分，可放这里

## 说明

当前这一步是“原始资料归档整理”，还不是“题目结构化入库”。
如果你下一步需要，我可以继续帮你把这些 Word 文件逐年抽取成：

- 题目文本
- 答案文本
- 选择题/填空题结构化数据
- 可直接导入项目的 JSON 题库
