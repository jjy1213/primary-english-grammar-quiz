# 题库图片目录

这个目录存放从原始 Word 试卷中导出的页面图片。

适用场景：

- 原始文档主要由扫描页或图片组成
- 直接转纯文本失败
- 需要后续 OCR 或人工辅助识别

## 当前内容

### `2020/`

从 `2020年上海市中考英语试卷和答案.docx` 中直接解包得到的页面图片资源。

目前已确认：

- 该文档正文几乎没有可抽取文本
- 主要内容存在于图片中
- 图片分辨率可用于后续 OCR 或人工校核

当前已确认：

- `image1.png` 是试卷封面信息
- `image2.jpg` 起是试题页面资源
- 这一年后续应优先走“图片识别/OCR”路线，而不是纯文本抽取

### `2018/` 和 `2021/`

这两份原始文件是 `.doc` 格式，不像 `2020.docx` 那样可以直接解包媒体资源。

当前状态：

- 已确认需要走“转换副本后再判断图片层/文字层”的路线
- 还没有完成中间 `.docx` 副本导出
- 因为 Word COM 转换较慢，后续建议单独逐份处理

## 后续建议

对图片版年份，推荐流程如下：

1. 提取页面图片
2. 按页编号整理
3. 做 OCR 或视觉识别
4. 再拆成题目文本、答案文本和结构化 JSON
